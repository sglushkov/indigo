# Talon6ROR driver

http://observatoriosspag.es/Whats_TALON6.htm

## Supported devices

* Talon6ROR

Single device is present on the first startup (no hot-plug support). Additional devices can be configured on runtime.

## Supported platforms

This driver is platform independent.
